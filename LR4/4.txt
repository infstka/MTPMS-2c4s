Builder
Builder - шаблон проектирования, инкапсулирующий создание объекта и позволяющий разделить его на разные этапы
Используется, когда:
- когда процесс создания нового объекта не должен зависеть от того, из каких частей этот объект состоит и как эти чачти связаны между собой 
- когда надо обеспечить получение различных вариаций объекта в процессе его создания

Singleton 
Singleton - паттерн, который гарантирует, что для определенного класса будет создан только 1 объект, а также предоставит к этому объекту точку доступа
Используется, когда:
- когда необходимо, чтобы для класса существовал 1 экземпляр
позволяет создать объект только при его необходимости

Prototype
Prototype - позволяет создать объекты на основе уже ранее созданныъ объектов-прототипов (т.е. клонирует объекты)
Используется, когда:
- когда конкретный тип создаваемого объекта должен определяться динамически во время выполнения
- когда клонирование является более предпочтительным вариантом, чем создание с помощью конструктора

Abstract Factory
АФ предоставляет интерфейс для создания семейств взаимосвязанных объектов с определенными интерфейсамим без указания конкретных типов данных объектов
Используется, когда:
- когда система не должна зависеть от способа создания и компоновки новых объектов 
- когда создаваемые объекты должны использоваться вместе  и являются взаимосвязанными
